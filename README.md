# Law and Beyond — WordPress Theme

A professional Indian legal news theme featuring Supreme Court, High Court, and legal updates. Magazine-style layout with top stories grid, category sections, and right sidebar.

## Features

- **Magazine-style homepage** — Top Stories hero grid + 5 configurable category sections
- **Realtime AJAX search** — Instant results as you type
- **Opinion & Monthly Recap sidebar** — Auto-populated from categories
- **Mobile bottom navigation** — Home, Latest, Search, SC, Menu
- **Push notification subscribe buttons** — Works with [LegalPress Push Notifications](https://github.com/CypherNinjaa/legalpress-push-notifications) plugin
- **Breadcrumbs, reading time, share buttons** — Built-in template tags
- **Live clock & date** — Red badge in top bar
- **Trending topics bar** — Below navigation
- **GitHub auto-updater** — Update the theme directly from WP Admin when new releases are available
- **Customizer settings** — Colors, social links, front-page sections, sidebar sections, copyright
- **Demo content importer** — One-click import of sample posts

## Requirements

- WordPress 6.0+
- PHP 7.4+

## Installation

1. Download the latest `.zip` from [Releases](https://github.com/CypherNinjaa/lawandbeyond/releases)
2. In WordPress Admin, go to **Appearance → Themes → Add New → Upload Theme**
3. Upload the `.zip` and activate
4. Go to **Appearance → Demo Content** to import sample posts

## Auto-Updates

This theme supports automatic updates from GitHub. When a new release is published:

1. Go to **Appearance → Themes**
2. You'll see an update notification on the theme card
3. Click **Update** to install the latest version

Or go to **Dashboard → Updates** to update along with plugins.
